Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install karma

Manual installation:

::

    $ git clone https://github.com/xeroc/python-karma/
    $ cd python-karma
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
